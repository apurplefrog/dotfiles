# apurplefrog's Dotfiles

[Screenshot of Environment](./screenshot.png)

- OS: NixOS
- WM: i3WM
- Compositor: picom
- Bar: polybar
- Terminal Emulator: kitty
- Text Editor: NeoVim
- Theme: Catppuccin Mocha

- Music Player: spotify-tui with spotifyd
- Fetcher: fastfetch
- Application Launcher: Rofi
